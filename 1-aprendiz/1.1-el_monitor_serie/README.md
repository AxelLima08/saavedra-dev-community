# El monitor serie  (sin solucionar)

**Dificultad:** 1.1 — Aprendiz  
**Tags:** `hardware` `arduino` `led` `aprendiz` `monitor serie`

---

## 📌 Consigna

Crea un programa que imprima en el monitor serie 'Hola mundo' una sola vez. En este ejercicio tocarás por primera vez el monitor serie. La idea es que hagas tu primer 'Hola mundo' y que se pueda visualizar en el monitor serie.

---

## ⚠️ Restricciones

- Solo se permite usar el framework de Arduino
- No se permite usar librerías externas
- Usar la placa arduino uno

---

## 💡 Hints

- ¡Ojo dónde ponés la función `Serial.println()`! Recordá la diferencia entre las dos secciones principales del código (`setup` y `loop`) si querés que el mensaje salga una sola vez.
